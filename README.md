# CalcRoll Information

電卓のようにキータップして計算式を入れてください。最後に[=]をタップすれば答えが出ます

以上が基本操作です

以下の機能が隠れていますので必要であれば直感的に操作して見つけてください

## 機能
    - 小数点の文字を選択できます（デフォルト：[.]ピリオド）

    - 小数の表示桁数が変更できます（デフォルト：3桁）
        
    - 小数の表示桁数で区切る場合の「丸め処理」が選択できます（デフォルト：五捨五超入）
        - 内部有効60桁処理により五捨五超入（偶数丸め）が正しく行えます

    - 桁区切りの文字を選択できます（デフォルト：[,]コンマ）

    - 桁区切りの方式を選択できます（デフォルト：3桁区切り）
        - インド式桁区切もできます

    - 計算機を増設して切替利用できます（デフォルト：1基）
    
    - 計算式と答えが記録されて参照や引用ができます
    
    - 計算式や答えの文字サイズを拡大縮小できます

    - 消費税込・税抜の計算ができます

    - 単位付きの計算ができます

    - キーボードをお好みにレイアウトできます

## 技術うんちく
    - 計算エンジン：　十進固定小数点方式、C/C++、最大有効6万桁超のうち60桁だけ使用
    - 計算式フォーム：　最大３フォーム、iPad等最大５フォーム
    - 履歴リスト：　最大100件、古いものから削除循環方式
    - 計算方式：　四則公式
    - 小数桁数：　0~10(最大15桁)
    - 端数丸め：　7種
        - 切り上げ
        - 偶数丸め(JIS丸め)
        - 切り捨て
    - 桁区切り文字：　3種
    - 小数点文字：　3種
    - 単位計算：
