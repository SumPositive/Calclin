# CalcRoll Information

## 特徴 

    - ロール紙に記録されるように計算式や結果が記録されます 

    - 複数の計算ができます (最大3ロール) 

    - 消費税や単位計算ができます 

    - キーボードをお好みにレイアウトできます 

    - 内部処理:60桁により正しい偶数丸めができます 

    - インド式桁区切りがあります 


## 詳細

    - 内部処理:有効60桁10進演算(固定小数点方式)

    - ドラム数:最大3

    - 計算方式:四則公式

    - 小数桁数:0~10(最大15桁)

    - 端数丸め:7種
        - 切り上げ
        - 偶数丸め(JIS丸め)
        - 切り捨て
        
    - 桁区切り:3種

    - 小 数 点:3種
